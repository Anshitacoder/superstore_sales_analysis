# Superstore Sales Analysis

##  Problem Statement
Businesses need to understand sales patterns to improve performance and decision-making. This project analyzes sales data to uncover key trends and insights.

##  Objective
To analyze sales data and identify patterns in product categories, regions, and time-based trends.

##  Tools Used
- SQL
- Python (Pandas)
- Power BI
- Excel

## Analysis Performed
- Category-wise order analysis
- Region-wise sales analysis
- Monthly sales trend analysis

##  Dashboard
(See dashboard.png uploaded in repository)

##  Key Insights

### Category Analysis
- Office Supplies has the highest number of orders  
- Technology has the lowest number of orders  
- This indicates that Office Supplies products are purchased more frequently  

### Regional Analysis
- West region has the highest sales among all regions  
- South region has the lowest sales  
- This indicates that the West region is the strongest market in terms of revenue  

### Sales Trend Analysis
- Sales show an upward trend over the months, indicating business growth  
- Early months have comparatively lower sales performance  
- Highest sales are observed in later months, suggesting seasonal demand or year-end purchasing behavior  

## Conclusion
The analysis highlights key sales patterns across categories, regions, and time. These insights can help businesses focus on high-performing areas and plan strategies for growth.
